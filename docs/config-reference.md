# 設定リファレンス（雛形）

| 変数 | 用途 | 例 |
|---|---|---|
| `NOTION_DATABASE_ID` | 投稿キューDBのID | `xxxxxxxxxxxxxxxxxxxx` |
| `NOTION_PROP_STATUS` | ステータスプロパティ名 | `Status` |
| `NOTION_PROP_POSTED_AT` | 投稿日時プロパティ名 | `PostedAt` |
| `STATUS_NOT_STARTED` | 未着手ラベル | `未着手` |
| `STATUS_PROGRESS` | 進行中ラベル | `progress` |
| `STATUS_READY` | 投稿準備OK | `ready` |
| `STATUS_POSTED` | 投稿済み | `posted` |
